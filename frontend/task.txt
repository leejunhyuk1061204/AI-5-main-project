# 프로젝트 진행 상황 요약

## 1. 연동 완료된 기능 (Connected)
- **인증 (Auth)**: 회원가입, 로그인, 프로필 관리 (`services/auth.ts`)
- **차량 (Vehicle)**: 차량 등록, 조회, 수정, 삭제 (`api/vehicleApi.ts`)
- **OBD**: 데이터 전송 및 디바이스 연결 관리 (`api/obdApi.ts`)

## 2. 연동 필요 기능 (Pending)
- **AI 진단**: 이미지/소리 진단 API 연동 (`AiController`)
- **주행 이력**: 주행 그래프 및 운전 습관 분석 연동 (`TripController`)
- **정비 관리**: 소모품 주기 관리 및 영수증 OCR 연동 (`MaintenanceController`)
- **기타**: 알림, 클라우드 인증, 미디어 관리
