# 하이모빌리티 데이터 통합 조회 이슈 (2026-01-23)

## 문제 상황
- **목표**: Odometer(주행거리) 외에 연료량, 배터리 상태, 타이어 압력 등 모든 차량 데이터를 한 번에 가져오기 위해 API 호출 방식 변경 시도.
- **시도 내용**:
  - 기존: `GET /v1/vehicle-data/autoapi-13/{vin}` (동작함, 하지만 Odometer만 수신됨)
  - 변경 시도: `POST /v1/vehicle-data` (Body에 `properties` 목록 포함)
- **결과**: `404 Not Found` 에러 발생.

## 원인 분석
- `POST /v1/vehicle-data` 라는 엔드포인트가 하이모빌리티 샌드박스 환경에 존재하지 않는 것으로 추정됨.
- 잘못된 API 주소를 사용했기 때문에 서버가 요청을 인식하지 못함.

## 해결 방안 (To-Do)

2. 정확한 주소와 요청 포맷(Payload)을 확인한 뒤 다시 적용해야 함.
