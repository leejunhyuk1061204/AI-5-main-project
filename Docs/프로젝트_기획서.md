> **"OBD-II 데이터 기반 AI 예지 정비 플랫폼"**

---

## 1. 프로젝트 개요

### 1.1 프로젝트 정보

| 항목 | 내용 |
|:---|:---|
| **프로젝트명** | 차량 예지 정비 시스템 (Project TBD) |
| **한줄 소개** | OBD-II 센서와 AI를 결합한 초개인화 차량 예지 정비 솔루션 |
| **목표** | 개인별 운전 습관과 실제 차량 상태를 반영한 데이터 기반 정비 예측 |
| **플랫폼** | 웹 / 모바일 앱 (반응형) |

### 1.2 핵심 가치

| 가치 | 설명 |
|:---|:---|
| **데이터 기반** | "단순히 오래됐으니까"가 아니라 "내 차 데이터가 이러하니까" |
| **예지 정비** | 고장이 나기 **전에** 미리 예측하고 대비 |
| **범용성** | 신차/구형차 관계없이 OBD-II 지원 차량 모두 사용 가능 |

### 1.3 핵심 기능

| 기능 | 설명 |
|:---|:---|
| **실시간 차량 진단** | OBD-II 데이터로 엔진/연료/배기 상태 모니터링 |
| **고장 코드 해석** | DTC 코드를 AI가 자연어로 쉽게 설명 |
| **셀프 점검 (멀티모달)** | 시동음 녹음, 부품 사진으로 AI 진단 ← **핵심 차별점** |
| **소모품 교체 예측** | 운전 습관 + 주행거리 기반 맞춤형 예측 |
| **리콜/검사 알림** | 공공 API 연동으로 리콜 및 정기검사 알림 |

## 4. 경쟁 서비스 분석

### 4.1 현재 서비스 특징

| 서비스 | 주요 기능 | 장점 |
|:---|:---|:---|
| **마이클 (Mycle)** | 주유/정비 기록, 정비소 예약 | 높은 인지도, 많은 제휴 정비소 |
| **카닥 (Cardoc)** | 외장 수리, 견적 비교 | 사진 업로드 → 견적 비교 편리 |
| **차계부 앱들** | 주유량, 비용 기록 | 심플한 UX |
| **제조사 앱** | 커넥티드카 기능 | 공식 데이터 연동 |

### 4.2 기존 서비스의 불편한점

| 문제점 | 상세 설명 |
|:---|:---|
| **❌ 수동 기록 의존** | 주유, 정비 내역을 사용자가 꾸준히 기록해야만 유용함 |
| **❌ 단순 주행거리 기준** | "5,000km마다 오일 교체" → 개인별 운전 습관 미반영 |
| **❌ 정비 품질 불일치** | 앱 점검 리포트와 실제 차량 상태가 다른 사례 보고 |
| **❌ 가격 적정성 불명확** | 제시된 가격이 정당한지 비전문가가 판단하기 어려움 |
| **❌ 구형차 지원 불가** | 커넥티드카 API 미지원 차량은 스마트 기능 이용 불가 |
| **❌ 불필요한 기능 과다** | 본연의 관리 기능보다 부가 기능이 많아 UX 저하 |

## 5. 프로젝트의 차별점

### 5.1 핵심 차별화 요소

| 기존 서비스 | 본 시스템 |
|:---|:---|
| 수동 기록 | ✅ **OBD-II 자동 연동** |
| 주행거리 기준 예측 | ✅ **운전 습관 + 실시간 데이터 기반 예측** |
| 커넥티드카만 지원 | ✅ **1996년 이후 모든 차량 지원** |
| 단순 기록 앱 | ✅ **AI 진단 + 예지 정비** |
| 텍스트 기반 진단 | ✅ **음성/사진 멀티모달 AI** |

### 5.2 기술적 차별화

| 기술 | 설명 |
|:---|:---|
| **OBD-II 실시간 분석** | RPM, 연료 트림, 온도 등 20+ 파라미터 모니터링 |
| **LSTM 이상 감지** | 고장 전 이상 패턴 조기 포착 |
| **XGBoost 수명 예측** | 소모품별 개인화된 교체 시점 예측 |
| **LLM 진단 리포트** | 기술 용어를 쉬운 자연어로 설명 |
| **공공 API 연동** | 리콜, 정비이력, 검사 정보 자동 조회 |
